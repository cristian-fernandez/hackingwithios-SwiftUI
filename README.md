# hackingwithios-SwiftUI
This is source code of Hacking with iOS: SwiftUI Edition of Paul Hudson, https://www.hackingwithswift.com/books/ios-swiftui

# SwiftUI Learning Projects
## Project List

### Project 1: WeSplit
- Learn the basics of SwiftUI with your first project
- Introduction to SwiftUI app structure
- Creating forms, adding navigation bars
- Modifying program state and binding
- Implementing user interface controls
- Creating views in a loop

### Project 2: Guess the Flag
- Build a game with stacks, images, and alerts
- Using stacks for view arrangement
- Working with colors, images, and buttons
- Displaying alert messages

### Project 3: Views and Modifiers
- Dive deep into Swift's rendering system
- Using SwiftUI views and modifiers
- View composition, custom modifiers, and containers

### Project 4: BetterRest
- Use machine learning to improve sleep
- Entering numbers with Stepper
- Selecting dates and times with DatePicker
- Working with dates
- Training a model with Create ML

### Project 5: Word Scramble
- Build a letter rearranging game with List
- Introducing List, loading resources from app bundle
- Working with strings, adding to a list of words
- Validating words with UITextChecker

### Project 6: Animation
- Spruce up your UI with springs, bounces, and more
- Creating implicit and explicit animations
- Animating bindings, controlling animation stack

### Project 7: iExpense
- Bring in a second view with this expense tracking app
- Using @State and @StateObject, sharing state
- Deleting items using onDelete()
- Storing user settings with UserDefaults

### Project 8: Moonshot
- Teach users about space history with scroll views, Codable, and more
- Using GeometryReader, ScrollView, NavigationLink
- Working with hierarchical Codable data

### Project 9: Drawing
- Use shapes, paths, colors, and more to create custom art for your app
- Drawing custom paths, shapes, and transformations
- Animating shapes and creating custom transitions

### Project 10: Cupcake Corner
- Build an app that sends and receives JSON from the internet
- Adding Codable conformance for @Published properties
- Sending and receiving Codable data with URLSession

### Project 11: Bookworm
- Use Core Data to build an app that tracks books you like
- Creating NSManagedObject subclasses
- Using Core Data, NavigationLink, alerts

### Project 12: Core Data
- Take an in-depth tour of how SwiftUI and Core Data work together
- Filtering @FetchRequest using NSPredicate
- One-to-many relationships with Core Data, SwiftUI

### Project 13: Instafilter
- Learn to link SwiftUI, UIKit, and Core Image in one app
- Importing and filtering images using Core Image

### Project 14: Bucket List
- Embed maps and make network calls in this life goals app
- Integrating MapKit with SwiftUI
- Using Touch ID and Face ID with SwiftUI

### Project 15: Accessibility
- Learn how to make your apps available to everyone
- Identifying views with useful labels
- Hiding and grouping accessibility data

### Project 16: Hot Prospects
- Build an app for conferences with tabs, context menus, and more
- Using TabView, TabItem, context menus, coordinators

### Project 17: Flashzilla
- Use gestures and haptics to build a learning app
- Incorporating gestures, vibrations, timers

### Project 18: Layout and Geometry
- Explore the inner workings of SwiftUI's layout system
- Alignment, GeometryReader, ScrollView effects

### Project 19: SnowSeeker
- Build an app for ski resorts that works great on iPad
- Side-by-side views, searching, adapting layouts
